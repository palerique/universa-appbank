Mantém Cliente

Descrição:

Como Gerente de Conta, é preciso manter os dados cadastrais dos clientes da instituição bancária, podendo executar as operações de inclusão, alteração ou exclusão. Deve ser verificado se o cliente não tem restrições no CADIN.

Restrições:

  • O CPF do cliente deve ser um CPF válido.
  • Não deve haver mais de um cliente com o mesmo CPF.
  • O CPF 02728594430 é o único CPF que há com restrição no Cadin.
  • O login do cliente deve ser um e-mail válido.
  • Não deve haver mais de um cliente com o mesmo login.

Critérios de Aceitação:

  • Quando o cliente estiver pendente junto ao Cadin, a situação do cliente deve ser Pendente, caso contrário, Ativo.
  • Quando realizada uma inclusão, alteração ou exclusão de um cliente, deve-se:
    • Notifica o Cadin - Cadastro de Inadimplentes sobre a atualização cadastral do cliente, informando o CPF e o motivo da atualização que pode ser:
      • Para inclusão - Credenciamento de cliente
      • Para alteração - Atualização Cadastral
      • Para exclusão - Descredenciamento de cliente

Observação:

  • Códigos de retorno do Cadin para notificação de atualização cadastral:
    • 0 - Notificação realizada com sucesso
    • 1 - Dados insuficientes para notificação ao Cadin
    • 2 - CPF inválido.
  • Códigos de retorno do Cadin para verificação que o cliente está com restrições:
    • 3 - Cliente com restrições no Cadin
    • 4 - Cliente sem restrições no Cadin
